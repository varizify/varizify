# 🚖 صندوق همیار رانندگان – مدیریت اعتبار بر اساس مشارکت

یک ابزار سبک و بدون نیاز به سرور برای مدیریت صندوق قرض‌الحسنه گروهی.  
با آپلود یک فایل CSV ساده، به طور خودکار **مجموع مشارکت هر عضو** محاسبه شده و بر اساس **ضریب اعتبار** (وابسته به میزان واریز)، حداکثر وام قابل دریافت هر شخص نمایش داده می‌شود.  
ایده‌آل برای گروه‌های رانندگان استیجاری، تعاونی‌ها و هر گروهی که می‌خواهد بر اساس **مشارکت واقعی** به افراد اعتبار بدهد.

## ✨ ویژگی‌ها

- **بدون نیاز به سرور** – فقط یک فایل HTML که در مرورگر اجرا می‌شود.
- **محاسبه هوشمند اعتبار** – بر اساس جمع واریزها و ضریب‌های از پیش تعریف شده.
- **رتبه‌بندی اعضا** – جدول مرتب بر اساس بیشترین اعتبار قابل دریافت.
- **جستجو و فیلتر** – جستجوی نام اعضا.
- **نمایش جزئیات تراکنش‌ها** – لیست همه واریزها همراه با تاریخ.
- **بارگذاری فایل CSV** – به سادگی فایل خود را آپلود کنید.
- **داده نمونه داخلی** – برای آشنایی سریع با سیستم.
- **طراحی واکنش‌گرا** – قابل استفاده در موبایل و دسکتاپ.

## 🧮 نحوه محاسبه اعتبار (ضریب مشارکت)

| مجموع مشارکت (تومان) | ضریب | حداکثر وام قابل دریافت |
|----------------------|------|--------------------------|
| کمتر از ۵,۰۰۰,۰۰۰    | ۰    | ۰ تومان (عدم صلاحیت)     |
| ۵,۰۰۰,۰۰۰ تا ۲۰,۰۰۰,۰۰۰ | ۲ | تا ۲ برابر مجموع مشارکت |
| ۲۰,۰۰۰,۰۰۰ تا ۵۰,۰۰۰,۰۰۰ | ۴ | تا ۴ برابر مجموع مشارکت |
| بیشتر از ۵۰,۰۰۰,۰۰۰  | ۶    | تا ۶ برابر مجموع مشارکت |

> **مثال:** شخصی با ۲۰ میلیون تومان مشارکت، ضریب ۴ را کسب کرده و می‌تواند تا **۸۰ میلیون تومان** وام دریافت کند.

## 📁 فرمت فایل CSV

فایل CSV باید شامل ستون‌های زیر باشد (ترتیب مهم نیست، اما نام ستون‌ها دقیقاً به همین صورت باشد):

| نام | مبلغ | تاریخ |
|-----|------|-------|
| علی رضایی | 15000000 | 1403/01/10 |
| سعید کریمی | 22000000 | 1403/01/12 |

- **نام** – نام یا شناسه عضو (رشته متنی)
- **مبلغ** – عدد به تومان (بدون کاما، می‌تواند اعشاری نباشد)
- **تاریخ** – اختیاری، هر رشته متنی (مثلاً ۱۴۰۳/۰۲/۰۱)

> می‌توانید از دکمه **دانلود نمونه CSV** در صفحه استفاده کنید.

## 🚀 نحوه استفاده

1. فایل `index.html` را روی کامپیوتر خود ذخیره کنید.
2. فایل را با مرورگر (Chrome، Firefox، Edge، ...) باز کنید.
3. یکی از روش‌های زیر را برای وارد کردن داده انتخاب کنید:
   - کلیک روی **بارگذاری دمو** برای مشاهده داده‌های آزمایشی.
   - کلیک روی **دانلود نمونه CSV**، پر کردن فایل با داده‌های واقعی و سپس آپلود آن.
   - کشیدن و رها کردن فایل CSV در قسمت مشخص شده.
4. جدول رتبه‌بندی و جزئیات تراکنش‌ها به‌طور خودکار به‌روز می‌شوند.
5. از کادر جستجو برای فیلتر کردن اعضا استفاده کنید.

## ⚙️ تنظیم ضرایب (اختیاری)

اگر می‌خواهید بازه‌ها یا ضرایب را تغییر دهید، فایل `index.html` را در یک ویرایشگر متن باز کنید و تابع `getCoefficient` را ویرایش نمایید:

```javascript
function getCoefficient(total) {
    if (total < 5_000_000) return 0;
    if (total < 20_000_000) return 2;
    if (total < 50_000_000) return 4;
    return 6;
}
```

عددها را مطابق نیاز گروه خود تنظیم کنید.
